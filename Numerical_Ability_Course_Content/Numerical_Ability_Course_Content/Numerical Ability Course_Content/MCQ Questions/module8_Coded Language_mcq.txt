CODED LANGUAGE - ADVANCED MCQs

1. In a code where each letter is replaced by the letter two places ahead, what is the code for RUBY?
A) TGWA
B) TWDZ
C) TWAZ
D) SVAZ
Answer: B

2. If A=1, B=2, ..., Z=26, what is the numeric sum for the coded word MIX when each letter is shifted three positions forward before conversion?

D) 68
