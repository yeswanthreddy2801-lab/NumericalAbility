Introduction

Odd Man Out and Analogies are key reasoning topics used in placement exams. They test your ability to recognize patterns, relationships, and differences among given elements.

Core Concepts

Odd Man Out

Find the element that does NOT follow the pattern.

Analogy

Relationship between A : B must match C : D

Solving Strategy

1. Identify pattern first

2. Compare numbers / letters / meaning

3. Check relationships (not values)

4. Eliminate wrong options quickly

Solved Examples

Odd Man Out: 3, 6, 9, 12, 14

Answer: 14 (not divisible by 3)

Analogy: Pen : Write :: Knife : ?

Answer: Cut

Common Mistakes

  • Focusing on values instead of patterns
  • Ignoring relationship type in analogy
  • Overthinking simple logic
