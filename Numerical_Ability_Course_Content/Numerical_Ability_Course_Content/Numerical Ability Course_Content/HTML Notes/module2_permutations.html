Permutations

Arrangement based counting where order matters

Learning Objectives

  • Understand arrangement concept
  • Apply permutation formula
  • Solve linear and circular arrangements
  • Handle repetition cases

Concept

Permutation means ARRANGEMENT of objects where ORDER matters. AB ≠ BA

Formula

nPr = n! / (n−r)!

Linear arrangement (all items): n!

Circular permutation: (n−1)!

Types of Permutations

  • Linear arrangement
  • Partial arrangement
  • Circular arrangement
  • Repetition-based arrangement

Solved Example

Arrange 5 people in 3 seats:

5P3 = 5! / 2! = 60 ways

Circular Permutation

n people in a circle = (n−1)! arrangements

Shortcut Tips

  • Check if order matters
  • Use nPr for selection + arrangement
  • Use (n−1)! for circles
  • Simplify factorials early

Common Mistakes

  • Using combination instead of permutation
  • Ignoring order importance
  • Wrong circular formula
  • Not simplifying factorials
