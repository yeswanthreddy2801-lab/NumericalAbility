Combinations

Selection based counting where order does NOT matter

Learning Objectives

  • Understand selection concept
  • Learn combination formula
  • Apply shortcut properties
  • Solve placement problems

Concept

Combination means selecting items where order does NOT matter. AB = BA

Formula

nCr = n! / (r!(n−r)!)

Important Properties

  • nC0 = 1
  • nC1 = n
  • nCr = nC(n−r)

Solved Example

5C2 = 10 ways

Shortcut Tips

  • Use symmetry: nCr = nC(n−r)
  • Reduce factorial before calculation
  • Choose smaller r value

Real-World Applications

  • Selecting committee members
  • Lottery ticket selection
  • Group formation
  • Sports team selection

Placement Tips

  • Use nCr only when order does NOT matter
  • Use symmetry property nCr = nC(n-r)
  • Simplify factorial before calculating
