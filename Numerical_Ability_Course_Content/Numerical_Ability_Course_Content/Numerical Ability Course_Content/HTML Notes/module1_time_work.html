Introduction

Time and Work problems measure the amount of work completed by one or more people or machines in a given time. Instead of working directly with days, we calculate the work done per day, called the work rate.

Important Formulas

Work = Rate × Time
One Day Work = 1 / Number of Days
Together = 1/x + 1/y
Time Together = xy / (x+y)
Efficiency ∝ Work
Efficiency ∝ 1 / Time

Concept Explanation

If a person finishes a work in N days, then one day's work is 1/N. To calculate work done by multiple people, add their one-day work. Example: A finishes in 10 days. B finishes in 15 days. Together 1/10 + 1/15 = 5/30 = 1/6 Therefore, they complete the work in 6 days.

Shortcut Tricks

  • Never add days directly.
  • Convert everything into one-day work.
  • Use LCM to avoid fractions.
  • Remember xy/(x+y) for two workers.
  • Higher efficiency means less time.

Solved Example

A completes a work in 12 days. B completes it in 18 days. LCM = 36 A = 3 units/day B = 2 units/day Together = 5 units/day Time = 36/5 = 7.2 days

Common Mistakes

  • Adding days instead of work rates.
  • Ignoring efficiency ratio.
  • Using incorrect LCM.
  • Not simplifying fractions.
