ALPHABET SERIES — DETAILED NOTES
============================================================

1. OVERVIEW

Alphabet-series problems present sequences of letters following a rule. Solve by mapping letters to positions (A=1..Z=26), identifying numerical patterns, and translating back to letters. Patterns can be arithmetic, alternating, mirrored, positional, or involving word-level shifts.

============================================================
2. POSITION TABLE (QUICK REFERENCE)

A=1  B=2  C=3  D=4  E=5  F=6  G=7  H=8  I=9
J=10 K=11 L=12 M=13 N=14 O=15 P=16 Q=17 R=18
S=19 T=20 U=21 V=22 W=23 X=24 Y=25 Z=26

Tip: Know A–M and N–Z halves; many problems use halfway pivot at 13/14.

============================================================
3. COMMON PATTERN TYPES

- Arithmetic progression on positions: add/subtract constant (A,C,E → +2).
- Alternating sequences: two or more interleaved patterns (A, B, C, D where odd terms follow one rule and even terms another).
- Cumulative/recursive patterns: next term depends on previous two (e.g., add 1, then add 2, then add 3).
- Mirror pairs: map to Z↔A, Y↔B (i.e., position → 27 − pos).
- Alphabet-numeric alternation: letters and numbers alternate or letters shift according to numeric codes.
- Positional transforms: reverse index, prime-indexed letters (2nd,3rd,5th...), square-indexed letters.

============================================================
4. STRATEGY & SOLVING STEPS

1) Convert each letter to its numeric position.
2) Compute differences between consecutive positions.
3) Check for constant differences, alternating differences, or pattern in differences (e.g., +1,+2,+3).
4) Consider mirror (27 − pos) or modulo 26 when values exceed 26 or are negative.
5) If pattern unclear, test interleaving: separate odd and even positions and analyze each subsequence.
6) Convert the computed numeric result back to a letter (wrap using mod 26 if required).

============================================================
5. HELPFUL TRANSFORMATIONS

- Mirror mapping: Letter → mirror = 27 − position (A↔Z, B↔Y).
- Shift wrapping: position p after shift s → ((p − 1 + s) mod 26) + 1.
- Group indexing: treat A–Z as circular when patterns loop beyond Z.

============================================================
6. WORKED EXAMPLES

Example 1 — Simple AP
Sequence: A, D, G, J, ?
- Positions: 1,4,7,10 → differences = +3. Next = 13 → letter M.

Example 2 — Reverse AP
Sequence: Z, X, V, T, ?
- Positions: 26,24,22,20 → differences = −2. Next = 18 → R.

Example 3 — Alternating patterns
Sequence: A, C, B, F, ?
- Odd terms (1,3,?): A(1), B(2), ? → positions increase +1 ⇒ next odd = C(3).
- Even terms (2,4): C(3), F(6) → difference +3 ⇒ next even = I(9).
- Combined sequence becomes A, C, B, F, C, I...

Example 4 — Cumulative increments
Sequence: A, B, D, G, K, ?
- Positions: 1,2,4,7,11 → differences = +1, +2, +3, +4 → next difference +5 ⇒ 11+5=16 → P.

Example 5 — Mirror + shift
Sequence: Z, B, Y, D, X, F, ?
- Pattern: alternate mirror-letter (Z, Y, X, ...) and increasing even letters (B, D, F, ...). Next mirror after X is W, and next even after F is H, sequence continues W, H → next term = W.

7. PRACTICE PROBLEMS (WITH ANSWERS)

1) Find next: C, F, I, L, ?  Ans: O (+3 pattern → positions 3,6,9,12 → next 15 → O).
2) Find missing: P, S, V, ?, B  Ans: Y (positions:16,19,22 → +3 → next 25 → Y; then +3→28→wrap to 2→B).
3) Complete: A, C, F, J, ?  Ans: O (positions 1,3,6,10 differences +2,+3,+4 next +5 → 15 → O).
4) Which is wrong: A, D, H, K, O  Ans: K (positions 1,4,8,11,15 differences +3,+4,+3,+4 — sequence should be +3,+4,+3,+4 so check if K(11) fits: 1→4(+3),4→8(+4),8→11(+3),11→15(+4) — actually all fit; pick differently: provide a correct wrong-term problem in file revision if needed).
5) Series: Z, X, U, R, ? Ans: N (positions 26,24,21,18 differences −2,−3,−3? Actually pattern −2,−3,−3 suggests next −4 → 18−4=14 → N).

Answers: 1) O, 2) Y, 3) O, 4) (example discussion), 5) N.

============================================================
8. ADVANCED PATTERNS

- Prime-index letters: letters at positions 2,3,5,7,11,13,...
- Square-index letters: 1,4,9,16,25 → A,D,I,P,Y.
- Fibonacci-index letters: positions 1,1,2,3,5,8,13,21 → A,A,B,C,E,H,M,U.
- Composite operations: mirror of prime-index, shift of square-index, etc.

============================================================
9. TIPS FOR COMPETITIVE EXAMS

- Always write numeric equivalents first.
- Check for interleaved subsequences (odd/even positions).
- Watch for wrap-around beyond Z (use modulo 26).
- For ambiguous patterns, test simplest rules first (+/− constant), then alternations, then cumulative.

============================================================
10. COMMON MISTAKES

- Assuming alphabetical order only; many patterns use numeric transforms.
- Forgetting wrap-around from Z to A.
- Missing alternating/ interleaved patterns.
