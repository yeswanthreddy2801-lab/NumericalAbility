INTRODUCTION

Coded Language is a reasoning topic where words or numbers are given in a coded form. You must identify the pattern and decode or encode the message.

============================================================

TYPES OF CODING

1. Letter Shifting
2. Number Coding
3. Substitution Coding
4. Word Pattern Coding
5. Mixed Coding

============================================================

LETTER SHIFTING

A → B → C → D ... Z

Example:
A + 1 = B
B + 2 = D

Used in shifting-based coding questions.

============================================================

NUMBER CODING

Letters are replaced by numbers.

Example:
A = 1, B = 2, C = 3

HELLO → 8 5 12 12 15

============================================================

SUBSTITUTION CODING

Words are replaced with other words.

Example:
Apple = Red Fruit
Then Apple is coded as Red Fruit.

============================================================

SHORTCUT TECHNIQUES

============================================================

SOLVED EXAMPLE 1

If A = 1, B = 2, then CAT = ?

C = 3, A = 1, T = 20

Answer: 3 1 20

============================================================

SOLVED EXAMPLE 2

If DOG = EPH, then CAT = ?

Each letter +1 shift

C → D
A → B
T → U

Answer: DBU
