2. INTRODUCTION — WHY THIS TOPIC MATTERS

Odd Man Out and Analogy questions test pattern-recognition and relational reasoning — core skills for aptitude tests, competitive exams, and interview screening. These questions reward methodical checking and speed. Practice reduces overthinking and improves accuracy.

============================================================
3. ODD MAN OUT — PATTERN CATEGORIES

A. Semantic / Meaning-based
- One word/item does not belong by meaning (e.g., cat, dog, horse, apple → apple).

B. Grammatical / Part-of-speech
- One item differs in grammatical class (noun vs verb/adjective).

C. Orthographic / Spelling-based
- Based on number of letters, vowels/consonants, letter patterns (palindrome vs non-palindrome).

D. Positional / Alphabetic
- Based on alphabetical positions, letter shifts, or distance from 'A'.

E. Numeric / Arithmetic properties
- Parity (odd/even), prime/composite, square/cube, multiples, digit-sum properties.

F. Structural / Visual or Shape-based
- For symbol sets or shapes: symmetry, number of sides, angles.

G. Mixed / Combined rules
- Some problems combine two or more properties (e.g., one item is numeric and prime while others are even composites).

============================================================
4. METHOD: A FAST CHECKLIST (for Odd Man Out)

1) Scan for obvious semantic differences (meanings, categories).
2) Check simple numeric properties: parity, prime/composite, multiples, squares.
3) Inspect spelling/letters: vowel count, length, alphabetical order.
4) If items are letters/symbols, map to alphabetical indices and check arithmetic patterns.
5) If two items share the same special property (e.g., palindromic), the remaining one is odd.
6) When multiple candidate rules exist, choose the one that explains most items with least complexity.

5. ODD MAN OUT — WORKED EXAMPLES

Example 1 — Words
Group: Apple, Orange, Mango, Carrot
- Semantic categories: fruits, fruits, fruits, vegetable ⇒ Odd = Carrot.

Example 2 — Numbers
Group: 5, 11, 17, 21, 23
- Check primes: 5(prime),11(prime),17(prime),21(composite),23(prime) ⇒ Odd = 21.

Example 3 — Letters
Group: A, C, F, G
- Positions: 1,3,6,7 — differences 2,3,1 (no simple AP). Check vowels? A vowel, others consonants ⇒ Odd = A.

Example 4 — Mixed arithmetic
Group: 8, 27, 125, 64, 24
- Recognize cubes: 8(2^3),27(3^3),125(5^3),64(4^3),24(not cube) ⇒ Odd = 24.

6. ANALOGIES — TYPES & HOW TO SOLVE

Definition: A : B :: C : D — find the relation A→B and apply it to C to get D.

Major relation types:
- Semantic/Meaning: Dog : Bark :: Cat : Meow
- Functional/Usage: Pen : Write :: Knife : Cut
- Part — Whole: Leaf : Tree :: Petal : Flower
- Degree/Quantity: Seed : Tree :: Cell : Organism
- Symbolic/Alphabetic: A : 1 :: B : 2 (position mapping)
- Grammatical: Run : Running :: Swim : Swimming (tense/affix change)
- Mathematical/Number operations: 3 : 9 :: 4 : 16 (square)

Solving steps:
1) Identify the simplest clear relation between A and B (meaning first).
2) Verify relation by checking if it uniquely maps A→B.
3) Apply same mapping to C and compute D.
4) If multiple relations possible, prefer the one that uses standard lexical/semantic or simple arithmetic transformation.

7. ANALOGIES — WORKED EXAMPLES

Example 1 — Semantic

- Relationship: object : activity performed with object.

Example 3 — Alphabetic mapping

8. PRACTICE SET — ODD MAN OUT (SOLUTIONS AFTER)

Q1) Cat, Dog, Cow, Car
Q2) 14, 21, 28, 36, 42
Q3) RADAR, LEVEL, CIVIC, HELLO
Q4) 2, 4, 8, 16, 31
Q5) Mango, Banana, Apple, Spinach

ANSWERS & EXPLANATIONS — ODD MAN OUT

Q1) Car — semantic: three are animals, one is vehicle.

Q2) 36 — check multiples of 7? 14(7×2),21(7×3),28(7×4),36(not 7×5),42(7×6) ⇒ 36 is odd.

Q3) HELLO — others are palindromes (RADAR, LEVEL, CIVIC) ⇒ HELLO is odd.

Q4) 31 — check powers of two: 2,4,8,16 are powers of two; 31 is not ⇒ odd.

Q5) Spinach — vegetables vs fruits ⇒ Spinach odd.

9. PRACTICE SET — ANALOGIES (SOLUTIONS AFTER)

Q1) Tree : Forest :: Star : ?
Q2) Smoke : Fire :: Cry : ?
Q3) 2 : 4 :: 3 : ?
Q4) Teach : Teacher :: Act : ?
Q5) Dog : Puppy :: Cat : ?

ANSWERS & EXPLANATIONS — ANALOGIES

Q1) Sky or Galaxy — relationship ambiguous; prefer Galaxy if 'group of stars' mapping (Star : Galaxy). If intended as 'many trees -> forest', then many stars -> galaxy/constellation.

Q2) Tears — functional/causal: smoke indicates fire; crying indicates tears.

Q3) 9 — likely square mapping: 2→4(2^2), 3→9(3^2).

Q4) Actor — role derivation: Teach → Teacher, Act → Actor.

Q5) Kitten — young form: Dog→Puppy, Cat→Kitten.

Note: Q1 demonstrates that some analogy stems can be ambiguous; when multiple answers fit, exams typically provide multiple choices that narrow the intended relation.

============================================================
10. STRATEGIES FOR AMBIGUITIES

- Prefer semantic or common dictionary relationships over contrived numeric transforms unless sequence shows numbers.
- Use the answer choices to disambiguate; eliminate choices that do not preserve the same relation.
- When stuck, reframe relation in plain English: "A is to B as C is to ?" and test candidate words.

============================================================
11. QUICK TIPS & TIME-SAVERS

- Scan for obvious outliers first; semantic differences are fastest.
- For letter/word problems, map letters to positions only if simpler options fail.
- For numeric groups, check parity and primality quickly (these are common odd-one-out rules).
- Under time pressure, eliminate at least two options quickly to improve odds.

============================================================
12. COMMON MISTAKES TO AVOID

- Overcomplicating: avoid inventing complex patterns when a simple category explains items.
- Ignoring index positions for interleaved lists.
- Forgetting spelling-based traps (e.g., silent letters, different plural forms).

============================================================
13. SUMMARY

• Start with meaning-based checks, then move to structural and numeric checks.
• For analogies, identify core relation and apply it uniformly.
• Use elimination, then verify the chosen rule fits all terms.
