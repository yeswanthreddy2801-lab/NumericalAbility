PERMUTATIONS - DETAILED NOTES

1. INTRODUCTION
Permutation counts the number of ways in which objects can be arranged when order matters. This concept is essential in arrangement problems, ranking problems, and ordering tasks.

2. KEY IDEA
If the order of selection or placement changes the outcome, use permutations. Permutations are about arrangements, not just selections.

3. PERMUTATION FORMULA
The number of ways to arrange r objects from n distinct objects is:
P(n, r) = nPr = n! / (n − r)!

Where:
- n is the total number of objects.
- r is the number of objects to arrange.

4. FULL PERMUTATION
When arranging all n objects, the number of permutations is:
nPn = n!.

5. UNDERSTANDING THE FORMULA
P(n, r) = n × (n − 1) × (n − 2) × ... × (n − r + 1).
This is the number of ordered arrangements of r out of n items.

6. PERMUTATIONS VS COMBINATIONS
- Permutation: order matters.
- Combination: order does not matter.

Example:
Arranging 3 books on a shelf = permutation.
Selecting 3 books to borrow = combination.

7. TYPES OF PERMUTATIONS
7.1 Linear permutations
Objects arranged in a row or line.
Example: seating people in a row.

7.2 Circular permutations
Objects arranged in a circle. Rotations are considered identical.
Number of ways = (n − 1)!.
Example: seating around a round table.

7.3 Permutations with repetition
When repetition is allowed, the count is:
n^r.
Example: forming 3-letter codes from 5 letters with repetition.

7.4 Permutations with identical objects
When some objects repeat, divide by the factorial of repeated counts:
Total arrangements = n! / (n1! × n2! × ...).
Example: arrangements of the word BALLOON = 7! / (2! × 2!).

8. SOLVED EXAMPLES
Example 1: Arrange books
How many ways can 4 books be arranged on a shelf?
Number of ways = 4! = 24.

Example 2: Selecting winners
From 6 candidates, how many ways can first, second, and third prizes be awarded?
P(6, 3) = 6 × 5 × 4 = 120.

Example 3: Circular arrangement
How many ways can 5 people sit around a round table?
Number of ways = (5 − 1)! = 24.

Example 4: Repetition allowed
How many 3-digit numbers can be formed using digits 1, 2, 3, 4 with repetition allowed?
Number of ways = 4^3 = 64.

Example 5: Identical items
How many arrangements of the letters in MISSISSIPPI?
Total letters = 11, with repeats: 4 S, 4 I, 2 P, 1 M.
Number of ways = 11! / (4! × 4! × 2! × 1!).

9. SHORTCUT FORMULAS
- nPr = n! / (n − r)!
- nPr = n × (n − 1) × ... × (n − r + 1)
- nPn = n!
- Circular permutation = (n − 1)!
- Repetition permutation = n^r

10. COMMON MISTAKES
- Using combination when order matters.
- Forgetting the subtractive term in nPr.
- Confusing circular permutation with linear permutation.
- Ignoring repeated objects in a set.
- Counting rotations as different in circular arrangements.

11. PRACTICE QUESTIONS
1. How many ways can 5 runners finish in first, second, and third place?
Answer: P(5, 3) = 60.

2. In how many ways can 4 letters be arranged from the word TRAIN?
Answer: P(5, 4) = 120.

3. How many arrangements are there for the letters of the word APPLE?
Answer: 5! / 2! = 60.

4. How many ways can 3 digits be chosen from 0–9 if repetition is allowed?
Answer: 10^3 = 1000.

12. APPLICATIONS
- Ranking competitors
- Scheduling tasks
- Arranging seats and objects
- Creating passwords and security codes
- Organizing lists and sequences

13. QUICK SUMMARY
- Permutations count ordered arrangements.
- Use P(n, r) when order matters.
- Use n! when arranging all objects.
- Use (n − 1)! for circular arrangements.
- Use n^r for repeated choices.
- Divide by repeated factorials when items are identical.

Permutations form the foundation of order-based counting problems. Practice both simple and restricted arrangement questions to build familiarity and speed.
