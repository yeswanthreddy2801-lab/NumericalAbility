DI PIE CHARTS (DETAILED NOTES)

============================================================
1. OBJECTIVE

Understand pie-chart construction and interpretation, convert between percentages/angles/values, solve typical DI problems (missing values, comparisons, compound sectors), and apply quick shortcuts for competitive exams.

============================================================
2. FUNDAMENTAL RELATIONSHIPS

- Full circle = 360° = 100% of the total.
- 1% = 3.6°; 1° = 100/360 ≈ 0.277777...%.
- Angle corresponding to a sector = (percentage × 360) / 100 = percentage × 3.6.
- Value corresponding to a sector = (angle / 360) × total = (percentage / 100) × total.

Always determine the total (sum represented by whole pie) before computing sector values.

============================================================
3. COMMON PROBLEM TYPES

- Given percentages and total → find sector values or angles.
- Given angles and total → find percentages and/or values.
- One or more sectors missing → infer missing percentages/values from the total = 100%.
- Ratio-based data → convert ratio to percentage, then to degrees/values.
- Compound questions: compare sums of sectors, find difference percentages/angles, overlapping sectors.

============================================================
4. USEFUL SHORTCUTS

- Convert percentage → degrees: multiply by 3.6 (or ×36 then ÷10).
- Convert degrees → percentage: multiply by (100/360) = 5/18 ≈ 0.27778.
- When percentages are multiples of 5 or 10, convert quickly via simple factors (e.g., 10% = 36°).
- For ratios a:b:c, first compute total parts = a+b+c; percentage for a = (a/total_parts)×100.
- Use proportional reasoning: if total changes, sector angles remain proportional to values.

============================================================
5. STEP-BY-STEP STRATEGY

1) Identify total (T). If not given, assume 100 (for % problems) or use a convenient total (LCM of denominators) for ratios.
2) Convert ratios/percentages to percentages or values.
3) Compute angle = percentage × 3.6 when needed.
4) For missing sectors, subtract known percentages from 100%.
5) Use elimination on MCQs; quick estimation can rule out incorrect choices.

============================================================
6. WORKED EXAMPLES

Example 1 — Basic conversion
Total production = 900 units. Sector A = 25%.
- Value = 0.25 × 900 = 225 units.
- Angle = 25 × 3.6 = 90°.

Example 2 — Missing sector
Pie sectors: A=40%, B=27%, C=18%, D=?
- D = 100 − (40+27+18) = 15%.
- If total = 1200, D = 0.15×1200 = 180; angle = 15×3.6 = 54°.

Example 3 — Ratio to angle
Sectors in ratio 3:2:5; total sales = 3600.
- Total parts = 10; part value = 360.
- Sector values = 1080, 720, 1800.
- Percentages = 30%, 20%, 50% → angles = 108°, 72°, 180°.

Example 4 — Comparative question
If A sector is 40% and B sector is 30% of the same total, what is A − B in degrees?
- Difference % = 10% → difference angle = 10×3.6 = 36°.

Example 5 — When total is unknown
Pie shows angles: A=120°, B=90°, C=150°. If value of A = 240, find total.
- A% = (120/360)×100 = 33.333...% ⇒ total = 240 / 0.33333 = 720.

============================================================
7. TRICKS FOR FAST CALCULATION

- When totals are round, convert percentages to direct fractions: 25% = 1/4, 20% = 1/5.
- Use 360 as base: instead of working in percentages, operate directly in degrees when given angles.
- For small adjustments, note 1% = 3.6° so shifting 1% changes angle by 3.6°.
- Use integer-friendly totals: when dealing with ratios with denominators like 3, 7, 11, pick totals equal to LCM to avoid fractions (e.g., total = 100 or LCM of parts).

============================================================
8. PRACTICE PROBLEMS (ANSWERS FOLLOW)

1) Total students = 800. Pie chart shows A=135°, B=90°, C=45°, D=? Find D's students.
2) Sectors are in ratio 4:5:7 and total = 3200. Find angle for the first sector.
3) If A = 20% and B = 25% of total sales, what is angle subtended by A+B?
4) Pie chart angles: 54°, 126°, 90°, 90°. If smallest sector's value = 48, find total.
5) A pie chart shows ages in five groups: if group1=10%, group2=15%, group3=20%, group4=25%, group5=?; find angle for group5.

Answers:
1) A=135°→(135/360)×800=300; B=90°→200; C=45°→100; D = 800−(300+200+100)=200.
2) Total parts = 4+5+7=16; first sector fraction = 4/16=1/4 ⇒ percentage =25% ⇒ angle = 90°.
3) A+B = 45% ⇒ angle = 45×3.6 = 162°.
4) Smallest = 54° ⇒ percentage = 54/360 = 0.15 ⇒ total = 48 / 0.15 = 320.
5) group5 = 100 − (10+15+20+25) = 30% ⇒ angle = 108°.

============================================================
9. COMMON MISTAKES TO AVOID

- Mixing up degrees and percentages (remember 1% = 3.6°).
- Using incorrect total when pie represents partial totals.
- Rounding too early; keep fractions exact until final step.
- Forgetting to check that sector percentages sum to 100%.

============================================================
10. ADVANCED PROBLEMS (TIPS)

- For multi-step DI questions combine pie-chart analysis with bar or table data—normalize totals first.
- When comparing two pies with different totals convert sectors to percentages before comparing.
- For percentage increases/decreases applied to a sector, recalculate sector percentage relative to new total.
