INTRODUCTION

A Pie Chart is a circular graph divided into sectors. Each sector represents a percentage of the total value. It is widely used in Data Interpretation questions in placement exams, banking exams, and competitive tests.

============================================================

BASIC CONCEPT

Total Circle = 360°

Total Value = 100%

1% = 3.6°

Formula:

Value = (Percentage / 100) × Total Value

============================================================

IMPORTANT POINTS

• Entire pie = 100%
• Each sector shows percentage share
• Larger sector = larger value
• Convert % into values for solving questions
• Always identify total value first

============================================================

SHORTCUT TECHNIQUES

• Convert % directly into fractions
• Use ratio comparison instead of full calculation
• Multiply only when required
• Avoid full decimal calculations
• Use elimination method in MCQs

============================================================

COMMON FORMULAS

1. Value = (Percentage × Total) / 100

2. Angle = (Percentage × 360) / 100

3. Percentage = (Value / Total) × 100

============================================================

SOLVED EXAMPLE

If total sales = 2000 and A = 25%

A's value = (25/100) × 2000 = 500

============================================================
